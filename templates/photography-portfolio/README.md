# Lena Vale Studio

A portfolio template for photographers and image-makers who need rigorous composition, project rhythm, and clean editorial restraint.

## Pages

- `index.html`
- `portfolio.html`
- `project-editorial.html`
- `about.html`
- `contact.html`

## Included Files

- `styles.css`
- `script.js`
- `assets/`
- `.github/workflows/deploy-pages.yml`

## Local Preview

```bash
python3 -m http.server 4321
```

Then open `http://127.0.0.1:4321`.

## Browser Review

```bash
node ../e2e/forced-browser-review.mjs --config ../e2e/configs/photography-portfolio.json
```

## Deployment

This template is GitHub Pages ready and uses relative paths so it can publish from a project subpath.
